Sistema de Gestión de Restaurante con Autenticación y Redirección por Roles

Este sistema de autenticación permite a los usuarios iniciar sesión y ser redirigidos automáticamente a diferentes rutas según sus roles. Es ideal para aplicaciones web que requieren una gestión diferenciada de usuarios con distintos niveles de permisos y responsabilidades. Los usuarios pueden tener uno o múltiples roles, y el sistema les proporcionará acceso específico basado en sus roles asignados.
